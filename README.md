Mobile App Summary
CampusFlow is the mobile application for the SVIT Hackathon project, built with Expo, React Native, and Expo Router. It provides a role-based academic management experience for students, teachers, and admins, bringing together attendance tracking, performance monitoring, announcements, academic records, and institutional workflows in a single mobile interface.

The app includes secure authentication with persistent sessions, role-aware navigation, and dedicated dashboards for each user type. Students can view attendance, marks, profile details, and updates; teachers can manage attendance sessions, class assignments, and performance data; admins get a college-wide command center for people management, academics, announcements, and high-level performance and attendance insights.

Key Highlights
• Expo + React Native mobile app with Expo Router
• Role-based access for Admin, Teacher, and Student
• Attendance management and tracking
• Performance and marks monitoring
• Announcements and academic updates
• Secure session handling with expo-secure-store
• State and data management using Zustand and React Query
